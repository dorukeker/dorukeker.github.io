# Bubble Fountain

## About

An experiment to understand p5.js partical system. It based on the example [here](https://p5js.org/examples/examples/Simulate_Particle_System.php)

## How to setup development environment

Make sure you have npm and bower installed

1. Check out the repo

2. Run `$ npm install` and `$ bower install`

3. Run `$ npm start` the sketch will be served on `http://127.0.0.1:4000`
